2667. Create Hello World Function

Easy Companies

Example 1:
Input: args = []
Output: "Hello World"
Explanation:

f(); // "Hello World"
                            
The function returned by createHelloWorld should always return "Hello World".

Input: args = [{},null,42]
Output: "Hello World"
Explanation:

Any arguments could be passed to the function but it should still always return "Hello World".
